Simple-Sentinel

A stable, and easy to use redis-sentinel client for Node.js.

View on GitHub Download .zip Download .tar.gz

Simple-Sentinel is a stable redis-sentinel library with a clean event-based API. It has modern features that let you reliably connect to a cluster of redis-sentinels, and get real-time updates whenever things change.

Get it through npm:

npm install --save simple-sentinel

Sample Code

// Put all sentinel servers in here. A random one will be used:
var sentinels = [
  { host: '10.2.3.41', port: 26379 },
  { host: '10.2.3.42', port: 26379 },
  { host: '10.2.3.43', port: 26379 },
  { host: '10.2.3.44', port: 26379 },
  { host: '10.2.3.45', port: 26379 }
];

// Create a sentinel object to track all available Replicas:
var sentinel = new RedisSentinel(sentinels);

// A place to find all Replica Sets that we find:
var masters = {};

// Listen for connection info changes:
sentinel.on('change', function (name, replica) {
  console.log("Just got connection info for Replica:", name, replica.toString());
  masters[name] = replica.connectMaster();
});
